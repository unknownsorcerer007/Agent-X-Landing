Quick Onboarding

Install and start running in less than two minutes.

Install all dependencies, virtual environments, offline solver libraries, and launch the setup wizard automatically:

# Open PowerShell and run the installer script 
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.SecurityProtocolType]::Tls12; iex ((New-Object System.Net.WebClient).DownloadString('https://raw.githubusercontent.com/unknownsorcerer007/Agent-X/main/install.ps1'))

Launch the full service suite including Redis, Postgres, and the Agent-X server instances in containers:

# Clone and pull the compose configuration 
docker compose up -d

Run the automatic installation and verification bash script:

# Download and execute setup.sh 
curl -sSL https://raw.githubusercontent.com/unknownsorcerer007/Agent-X/main/setup.sh | bash
